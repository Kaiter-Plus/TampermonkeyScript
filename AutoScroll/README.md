## 1 功能介绍

给网页添加自动向上、向下滚动的按钮。对于一些经常需要滚动的场景很有用，比如看文章，看小说，看漫画等场景

### 1.1 如何使用它

使用很简单，直接使用【鼠标指针】移动覆盖到相应的按钮上即可使网页自动滚动，【鼠标指针】移动覆盖到【中间数值】，然后使用【鼠标滚轮】滚动即可调整滚动速度
